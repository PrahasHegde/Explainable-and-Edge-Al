## Overview

This repository contains a comprehensive portfolio showcasing professional work, technical expertise, and career highlights. It serves as a central hub for demonstrating skills, completed projects, and professional achievements.

The CV Portfolio 2 project is designed to be:
- **Professional**: Clean, polished presentation of work and qualifications
- **Accessible**: Easy to navigate and understand for potential employers and collaborators
- **Comprehensive**: Detailed documentation of projects, skills, and experience
- **Maintainable**: Well-organized structure for easy updates and management

## Installation

### Prerequisites

- Git
- A web browser (for viewing portfolio)
- Optional: Node.js and npm (if the project includes build scripts)

### Clone the Repository

```bash
git clone https://github.com/PrahasHegde/cv_portfolio_2.git
cd cv_portfolio_2
```

### Optional: Install Dependencies

If the project includes npm dependencies:

```bash
npm install
```

## Usage

### Viewing the Portfolio

1. **Online Access**: Visit the hosted portfolio (link to be added when available)

2. **Local Viewing**: 
   - Open the `index.html` file directly in your web browser
   - Or use a local server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js
     npx http-server
     ```
   - Then navigate to `http://localhost:8000`

3. **Reading the CV**:
   - PDF version available in the `CV/` directory
   - Additional documentation in the `docs/` folder

### Updating the Portfolio

To update your portfolio:

1. Edit relevant files in the project structure
2. Update `CV/cv.pdf` with latest information
3. Add new projects to the `projects/` directory
4. Commit and push changes:
   ```bash
   git add .
   git commit -m "Update portfolio with new projects/skills"
   git push origin main
   ```

## Getting Started

### For First-Time Visitors

1. Start by reading this README to understand the project structure

3. Explore individual project folders for detailed case studies
4. Check the `docs/` folder for comprehensive skill and experience breakdowns
5. Download the CV from the `CV/` directory for a formal document
